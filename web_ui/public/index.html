🚀 极简流程：
  1. 安装依赖：pip install ultralytics roboflow torch torchvision
  2. 进入本项目 cd web_ui
  3. 一键训练：python local_training.py
  4. 所有结果自动输出到 nutriscan_training/
详细教程可见 LOCAL_TRAINING_GUIDE.md
原Colab按钮仅供迁移或参考，推荐本地生产训练！

  "dataset": "马来西亚食物数据集",

  "img_size": 640
}
